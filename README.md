## What is DashLook?

macOS users have had Quick Look for years — select a file, press Space, done. Windows never had this built-in. **DashLook** gives it to you.

It sits quietly in your system tray, and the moment you press Space on any file in Explorer, a beautiful preview window appears — instantly, without opening any app.

---

## Features

- 🚀 **Instant preview** — press `Space` in File Explorer, preview appears in under a second
- 🖼️ **Image viewer** — zoom in/out with mouse wheel, checkerboard transparency background
- 🎬 **Video player** — play/pause, seek bar, volume control, fullscreen support
- 🎵 **Audio player** — plays MP3, FLAC, WAV, AAC and more with album art display
- 📕 **PDF viewer** — full PDF rendering, scroll through pages, zoom
- 💻 **Code viewer** — syntax highlighting for 100+ languages with line numbers
- 📝 **Markdown viewer** — renders MD files as beautiful styled HTML with dark theme
- 📦 **Archive viewer** — browse ZIP, RAR, 7Z contents without extracting, with search/filter
- 🔤 **Font previewer** — see the font rendered at multiple sizes with full alphabet
- 🌐 **HTML viewer** — renders HTML/SVG files in a Chromium engine
- 🌙 **Dark theme** — elegant dark UI that matches Windows 11 style
- 🔌 **Plugin system** — add support for any new file type by dropping a DLL in the Plugins folder
- ⚡ **Starts with Windows** — runs in system tray automatically on login
- 🪶 **Lightweight** — tiny memory footprint, no background scanning
- ⌨️ **Keyboard friendly** — everything controllable from keyboard
- 🔁 **Toggle preview** — press Space again or Esc to close
- 📂 **Works everywhere** — File Explorer, Save/Open dialogs, and most file managers

## How to Use

Open **File Explorer**, click any file, press `Space`.

| Key / Action | What it does |
|---|---|
| `Space` | Open or close the preview |
| `Esc` | Close the preview |
| `Enter` | Open the file with its default app |
| Mouse wheel | Zoom in / out (images) |
| Click play | Play or pause (video/audio) |
| Drag window | Move the preview anywhere |
