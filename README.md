## What is DashLook?

macOS users have had Quick Look for years — select a file, press Space, done. Windows never had this. **DashLook** gives it to you.

It lives in your system tray and gives you instant previews of images, videos, PDFs, code files, archives, fonts, and more — without opening anything.

---

## Supported Files

| Type | Formats |
|------|---------|
| 🖼️ Images | JPG, PNG, GIF, BMP, WebP, ICO, TIFF, SVG |
| 🎬 Video | MP4, MKV, AVI, MOV, WMV, FLV, WebM |
| 🎵 Audio | MP3, FLAC, WAV, AAC, OGG, M4A, WMA |
| 📕 PDF | PDF documents |
| 💻 Code | C#, Python, JS, TS, Go, Rust, C++, HTML, CSS, JSON, YAML and 90+ more |
| 📝 Markdown | MD, MDX files with full rendering |
| 📦 Archives | ZIP, RAR, 7Z, TAR, GZ — browse contents without extracting |
| 🔤 Fonts | TTF, OTF, WOFF — preview with sample text |
| 🌐 Web | HTML, HTM, SVG files |

## Download & Install

Go to **[Releases](https://github.com/itrabbi24/DashLook/releases/latest)** and pick one:

| Option | File | Best for |
|--------|------|----------|
| **Portable** | `DashLook-x.x.x-portable-win-x64.zip` | Try it without installing |
| **Installer** | `DashLook-Setup.msi` | Permanent install + auto-start on login |

> **Requirements:** Windows 10 or Windows 11 (64-bit)

---

## How to Use

Just open **File Explorer**, click on any file, and press `Space`.

| Key | Action |
|-----|--------|
| `Space` | Open preview / Close preview |
| `Esc` | Close preview |
| `Enter` | Open file with default app |
| Mouse wheel | Zoom in/out (images) |

That's it.
