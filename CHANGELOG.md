# Changelog

## [1.1.0] - 2026-03-17

### Added
- folder preview support with summary cards
- recent preview history in the system tray
- startup fade banner while DashLook loads in the tray
- branded preview loader with logo and progress animation
- pin-on-top preview control

### Improved
- much stronger desktop and File Explorer selection tracking
- light theme contrast and tray readability
- cleaner preview title bar spacing and controls
- user-facing README focused on features and usage

### Fixed
- built-in viewer plugins now ship correctly with published Windows builds
- desktop selection no longer reopens stale files
- selection logs no longer flood on every poll tick
- update dialog actions now behave better in light theme
